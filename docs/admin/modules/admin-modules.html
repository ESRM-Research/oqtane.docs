    Admin Modules

    In the default installation of Oqtane, there are numerous modules classified under 'Admin Modules'. These modules are used on the pages for host or site administration and can be included on other pages as desired by someone with the Administrator or Host Users role.

    The admin modules include the following.

    • Dashboard - The Admin dashboard
    • Error - Displays an error message
    • Files - File management
    • Jobs - Scheduled jobs
    • Languages - Language management
    • Login - Show login button or login status if someone is already logged in
    • Logs - Event Logs
    • ModuleDefinitions -
    • Modules - Module management
    • Pages - Page management
    • Profiles - Profile management
    • RecycleBin - Recycle bin
    • Register - Message about status of registration for the site
    • Reset - For resetting password
    • Roles - Role management
    • Site - Site Settings
    • Sites - Site management
    • Sql - Create SQL queries against the database
    • SystemInfo - System info
    • Themes - Theme management
    • Upgrade - System update
    • UrlMappings - URL mappings
    • UserProfile -
    • Users - User management
    • Visitors - Visitor management
